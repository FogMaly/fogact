## 📝 架构说明

### 访问流程

```
用户访问 http://localhost:34010/
    ↓
web-server.js 返回 /frontend/index.html
    ↓
浏览器执行 meta refresh
    ↓
自动跳转到 http://localhost:34010/user/
    ↓
web-server.js 返回 /frontend/user/index.html
    ↓
Vue.js 应用加载
    ↓
请求 assets 资源文件
    ↓
web-server.js 提供静态文件服务
    ↓
应用完全加载，显示监控界面
```

### 前端技术栈

- **框架**: Vue.js 3
- **图表**: Chart.js
- **样式**: CSS Variables + 响应式设计
- **主题**: 深色/浅色模式切换
- **存储**: localStorage
- **构建**: Vite（已构建完成）

## ⚠️ 注意事项

### 1. API 端点配置

前端中的 API 调用可能需要配置后端地址：
- 当前可能指向 `https://api.fogact.com` 或 `https://localhost:34020`
- 需要根据实际部署环境调整 baseURL
- 检查 `assets/index-Da98HOxL.js` 中的 API 配置

### 2. 主题存储键

前端使用 `localStorage.fogact_theme` 存储主题设置：
- 可选值: `'light'`, `'dark'`, `'system'`
- 如需更改键名，需修改源码并重新构建

### 3. 端口配置

- 默认端口: 34010
- 可通过环境变量修改: `PORT=8080 node bin/web-server.js`
