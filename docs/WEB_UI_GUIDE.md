# Web UI 使用指南

## 启动 Web 界面

### 方法 1: 使用 npm 脚本（推荐）
```bash
cd /opt/cliproxy-activator
npm run web
```

### 方法 2: 直接运行
```bash
node /opt/cliproxy-activator/bin/web-server.js
```

### 方法 3: 全局安装后使用
```bash
npm install -g .
cliproxy-web
```

## 访问界面

启动后，在浏览器中打开：
```
http://localhost:3000
```

## 界面功能

### 1. 服务选择
- 下拉菜单选择 Claude Code 或 Codex

### 2. 激活码输入
- 输入框支持粘贴激活码
- 自动验证必填项

### 3. 激活按钮
- 点击后显示加载动画
- 自动验证激活码
- 自动测试节点
- 显示激活结果

### 4. 结果显示
- 成功：绿色提示框，显示服务、节点、激活码信息
- 失败：红色提示框，显示错误原因
- 包含 CLI 命令提示

## 界面特性

✨ **设计特点**
- 现代化渐变紫色背景
- 白色卡片式布局
- 圆角阴影效果
- 响应式设计（支持手机/平板）

🎯 **交互体验**
- 按钮悬停动画
- 加载旋转动画
- 表单验证提示
- 实时错误反馈

🌐 **API 集成**
- 自动调用 yunyi.cfd API
- 验证激活码有效性
- 获取可用节点列表
- 显示最优节点

## 使用流程

1. **启动服务器**
   ```bash
   npm run web
   ```

2. **打开浏览器**
   - 访问 http://localhost:3000

3. **选择服务**
   - 从下拉菜单选择 Claude Code 或 Codex

4. **输入激活码**
   - 粘贴或输入激活码

5. **点击激活**
   - 等待验证和节点测试

6. **查看结果**
   - 成功：显示配置信息和 CLI 命令
   - 失败：显示错误信息

7. **完成配置**
   - 按照提示在终端运行 CLI 命令完成配置

## 技术栈

- **前端**: 纯 HTML + CSS + JavaScript
- **后端**: Node.js HTTP 服务器
- **API**: Fetch API 调用 yunyi.cfd
- **样式**: 现代 CSS3（渐变、动画、响应式）

## 端口配置

默认端口：`3000`

如需修改端口，编辑 `bin/web-server.js`：
```javascript
const PORT = 3000; // 修改为其他端口
```

## 故障排除

### 端口被占用
```bash
# 查找占用端口的进程
lsof -i :3000

# 或使用其他端口
# 修改 bin/web-server.js 中的 PORT 变量
```

### 无法访问
- 检查服务器是否正常启动
- 确认防火墙设置
- 尝试使用 127.0.0.1:3000

### API 调用失败
- 检查网络连接
- 确认 yunyi.cfd 可访问
- 查看浏览器控制台错误信息
