### 6. 使用方式

**CLI 模式**
```bash
npm start                                    # 交互式菜单
npm start activate --service codex --code XXX  # 直接激活
npm start test                               # 测试节点
npm start restore                            # 恢复备份
```

**Web 模式**
```bash
npm run web          # 启动 Web 服务器
../scripts/START_WEB.sh       # 使用启动脚本
```

访问：http://localhost:34010

**测试**
```bash
npm test             # 运行测试套件
```
