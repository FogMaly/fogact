# CLIProxy Activator - 管理后台登录快速指南

## 🔐 登录访问

### 1. 启动/重启服务

```bash
# 重启服务以应用登录功能
../scripts/重启前端服务.sh
```

### 2. 访问登录页面

```
http://localhost:34010/admin-login.html
```

### 3. 输入密码

```
默认密码: admin123
```

### 4. 访问管理中心

登录成功后自动跳转到:
```
http://localhost:34010/management/
```

## 🔑 修改密码

```bash
# 临时修改（本次启动有效）
ADMIN_PASSWORD=your_password ../scripts/重启前端服务.sh

# 永久修改（添加到环境变量）
echo 'export ADMIN_PASSWORD=your_password' >> ~/.bashrc
source ~/.bashrc
../scripts/重启前端服务.sh
```

## 📍 所有访问地址

| 页面 | 地址 | 权限 |
|------|------|------|
| 用户前端 | http://localhost:34010/ | 公开 |
| API 监控 | http://localhost:34010/user/ | 公开 |
| 登录页面 | http://localhost:34010/admin-login.html | 公开 |
| 管理中心 | http://localhost:34010/management/ | 需登录 |

../scripts/停止前端服务.sh

## ⚠️ 重要提示

1. **首次使用**: 必须运行 `../scripts/重启前端服务.sh` 应用登录功能
2. **默认密码**: `admin123` 仅用于开发，生产环境请修改
3. **Session 有效期**: 24小时，过期需重新登录
4. **服务重启**: Session 会丢失，需重新登录

## 🔒 安全建议

- ✅ 修改默认密码
- ✅ 使用强密码（字母+数字+符号）
- ✅ 生产环境启用 HTTPS
- ✅ 定期更换密码
