# 🚀 快速启动指南

## 立即开始使用 FogAct

### 方式 1: CLI 命令行模式

```bash
cd /opt/fogact
npm start
```

然后按照交互式菜单操作。

### 方式 2: Web 图形界面模式

```bash
cd /opt/fogact
npm run web
```

启动后会显示所有访问地址：
- 本地: http://localhost:34010/
- 局域网: http://10.0.66.2:34010/
- 外网: http://YOUR_PUBLIC_IP:34010/

### 方式 3: 直接激活

**激活 Codex:**
```bash
npm start activate --service codex --code K1DHPY3P-4B2W-F1A4-DC4P-Y74TCQZXPNYT
```

**激活 Claude:**
