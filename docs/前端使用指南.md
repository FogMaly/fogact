# CLIProxy Activator - 快速使用指南

## 🚀 前端服务管理

### 启动服务

```bash
# 方式 1: 使用启动脚本（推荐）
../scripts/启动前端服务.sh

# 方式 2: 使用 npm
npm run web

# 方式 3: 直接运行
node bin/web-server.js
```

### 停止服务

```bash
# 使用停止脚本
../scripts/停止前端服务.sh

# 或手动停止
pkill -f "node.*web-server.js"
```

### 测试访问

```bash
# 运行自动化测试
../scripts/测试前端访问.sh
```

## 🌐 访问地址

- **本地访问**: http://localhost:34010/
- **网络访问**: http://<服务器IP>:34010/

访问根路径会自动跳转到 `/user/` 监控界面


## 🔧 常见问题

### 端口被占用

```bash
# 查看端口占用
lsof -i :34010

# 使用其他端口
PORT=8080 node bin/web-server.js
```

### 服务无法启动

```bash
# 检查日志
cat logs/web-server.log

# 确认 Node.js 版本
node --version  # 需要 >= 16.0.0
```

### 前端资源 404

```bash
# 确认文件存在
ls -la frontend/user/assets/

# 重新复制资源
cp -r /opt/fog-activator/frontend/user/assets /opt/cliproxy-activator/frontend/user/
```
